# Development

### Launch dev container on vscode

Open vscode's command palette :

`Cmd + Shift + p`

Then find and select :

`Dev Containers: Open Folder In Container`

Wait for docker to build and run the dev container...\
Everything should be set and installed at the end.

## Backend

Run go echo server

```bash
cd go
go run main.go

```

## Frontend

In another terminal session, let's run our front app

```bash
pnpm run dev
```
